◆ AddOneFromConsole()

int CourseSet::AddOneFromConsole ( )

Añade un curso en el conjunto.

Precondición
Hay preparados en el canal estandar de entrada el número de sesiones del curso + los strings de todas las sesiones respectivas al curso.
Postcondición
Si en el nuevo curso hay intersección de problemas entre sesiones, no se creará el curso. Retornará -1 Si no hay intersección de problemas entre sesiones, se creará el curso. Dentro del curso rellenaremos el curso_sesion_map, con key id_problema(string), y value id_sesion(string). Después de añadirlo, el identificador del curso será el tamaño del CourseSet. Retornará 0 (Ok)
