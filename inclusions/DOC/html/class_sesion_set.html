◆ AddOneFromConsole()

int SesionSet::AddOneFromConsole ( string  id_sesion)

Añade una sesion en el conjunto.

Precondición
El string id_sesion no es vacío y hay preparados en el canal estandar de entrada un número = P de identificadores de problemas + P identificadores (id_problems).
Postcondición
Si el id_sesion ya estaba en el parámetro implícito, pedirá los id_problemas pero no creará el BinTree ya que esta sesión será desechada. Retornará -1. si el id_sesion no existia en el parámetro implícito, pedirá los id_problemas para construir el BinTree respectivo. Retornará 0 (OK).
